Description

— Drawercli (Aurora version) is an small installer script that allows all user-applications can be extracted into readable-files for Termux Widget apps.

— This script is designed for Termux and Termux:Widget to display the list of user applications, so they can be executed via the Termux Widget.

Requirements

  • Termux
  • Termux:Widget

Prerequisites

Make sure the following packages are installed in Termux:

  • aapt
  • curl
  • wget
  • Standard utilities containing awk, sed, grep (e.g., coreutils, busybox, etc.)
