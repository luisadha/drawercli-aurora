There are several installation methods available.

To update your apps list, rerun code below.

Inspect & Install

Your trust is important to us; you can verify the source code first before proceeding with the installation.

Advanced install

The best alternative, if you’d rather not use the usual copy-paste method, is to use the “Share URL” option it just needs a bit of setup first to make this button work.

NOTE: For the Share link to Termux button to work correctly, you need to replace the contents of the file ~/bin/termux-url-opener with the code shown under the heading Simulate your code. This also applies to the other button.

📖 Usage

Go to your home screen and add a widget. Scroll down until you find the widget named Termux and add it to your home screen.

🛠️ Requirements

Make sure the following applications are available in your phone:

  • Termux apps
  • Termux:Widget apps

📦 Dependencies

Make sure the following packages are installed in Termux:

  • aapt
  • curl
  • wget
  • Standard utilities containing awk, sed, grep (e.g., coreutils, busybox, etc.)
